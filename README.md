# Expression-evaluator

Консольное приложение на Java для проверки корректности математических выражений и вычисления их значения.

Программа принимает выражение в инфиксной форме (обычная математическая запись), выполняет валидацию, преобразует его в обратную польскую нотацию (RPN) и вычисляет результат.  
В случае некорректного выражения выводится сообщение об ошибке.

---

### Поддерживаемые элементы выражения

- Целые числа
- Арифметические операции:
  - `+` — сложение
  - `-` — вычитание
  - `*` — умножение
  - `/` — деление
- Круглые скобки `(` `)`
- Пробелы (игнорируются при обработке)

### Обработка ошибок

Программа выявляет следующие типы ошибок:

- Несбалансированные скобки
- Оператор перед ")"
- Оператор после "("
- Пустые скобки
- Недопустимые символы
- Два оператора подряд
- Два числа подряд (случай если между цифрами числа введен пробел)
- Оператор в начале или в конце выражения
- Деление на ноль (на этапе вычисления)

---

## Архитектура решения

Input → Validation → Tokenization → RPN Conversion → Evaluation → Result

Решение построено как последовательный pipeline из независимых компонентов:

### Компоненты

#### 1. ExpressionTokenizer
Преобразует входную строку в последовательность токенов:
- числа
- операторы
- скобки
  
#### 2. ParenthesesValidator
Проверяет корректность расстановки скобок с использованием стека.

#### 3. ExpressionSyntaxValidator
Проверяет корректность синтаксиса:
- допустимые символы
- порядок операторов
- корректное расположение чисел и скобок

#### 4. ExpressionRPNConverter
Реализует алгоритм сортировочной станции (Shunting-yard algorithm) для преобразования инфиксной записи в обратную польскую нотацию.

Учитывает:
- приоритет операторов
- левую ассоциативность
- обработку скобок

#### 5. RpnEvaluator
Вычисляет выражение в формате RPN с использованием стека.

## Тестирование

Проект покрыт unit-тестами для ключевых компонентов и интеграционным тестом полного сценария вычисления.

### Набор тестов

- `ParenthesesValidatorTest` — проверка корректности скобок
- `ExpressionSyntaxValidatorTest` — проверка синтаксиса выражения
- `ExpressionTokenizerTest` — проверка разбиения выражения на токены
- `ExpressionRPNConverterTest` — проверка преобразования инфиксной записи в RPN
- `RpnEvaluatorTest` — проверка вычисления выражений в формате RPN
- `FullCalculatorTest` — интеграционный тест полного пайплайна (валидация → токены → RPN → вычисление)
